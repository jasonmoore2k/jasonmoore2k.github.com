sola bible

DownloadWindows Phone Store

Sola Bible is focused on being a sleek and efficient way to read and search the bible, providing free access to the word of God for everyone.

It currently requires your data connection for the ESV, NET and a few other translations. All public domain versions, such as KJV and a few others, are now available without using your data connection. More translations will become available as copyrights permit.

FAQ: Why aren't there more Bible translations?

Translations are the intellectual property of the publishers and license holders. This application honours all of these copyrights, and so it will only be able to include access to translations where permissions have been granted. At this stage, this does not include many that users have been requesting - such as HCSB, NIV, NKJV, NLT and RSV. If and when these permissions have been granted, expect to see them included in Sola. Until then, be patient or feel welcome to use one of the other bible reading apps available (although please make sure you choose the ones that uphold copyright law).

Versions that require data connection:

  • amp - Amplified Bible
  • cev - Contemporary English Version
  • emphbbl - Emphasized Bible
  • esv - English Standard Version
  • gnb - Good News Bible
  • gnt - Good News Translation
  • leb - Lexham English Bible
  • nasb - New American Standard Bible
  • net - New English Translation

Version that are public domain (no data connection required):

  • akjv - American King James Version
  • asv - American Standard Version
  • darby - Darby Bible
  • douayrheims - Douay-Rheims Bible
  • kjv - King James Version
  • rv - Revised Version
  • web - World English Bible
  • webster - Webster's Bible
  • ylt - Young's Literal Translation

changes

v1.11 - Added an import function (found on the application bar of the contents page) for loading in custom XML bible files.

v1.10 - Audio playback now uses the new background worker for playback outside of the application. This also fixes a bug in the previous version. UX improvements in the Read section to help moving between favourites and the passage view.

v1.9 - Improvements to the responsiveness of the application.

v1.8 - All public domain versions (currently 9) are now available offline. Smoother animations. Prayers and favourites removed via context menu. A 'what's new' section showing the history of changes, and a 'legal' section for recognition of the other libraries used in the software. Search results link to the chapter text. Notes (by chapter) can be saved to an Evernote account.

v1.7 - A place to note personal prayer and thanksgiving, share these publicly, additional reading plans and streaming audio for the ESV version.

v1.6 - Few additional translations from http://www.4-14.org.uk and a search function via the http://biblia.com web service.

v1.5 - Minor improvements to the daily reading section and selection lists.

v1.4 - Added a daily bible reading plan. Currently for reading the bible over the course of a year, but stayed tuned for plans that go at a slower pace.

v1.3 - Allows the application to come back a lot faster from the lock screen. Chapters can be removed from the favourites.

v1.2 - UI redesigned and geared towards making the jump between passages as quick as possible. Chapters can be added to favourites for quick access. Can now also choose from ASV, KJV, AKJV and WEB translations.
