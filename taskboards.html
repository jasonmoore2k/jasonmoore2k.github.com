taskboards

DownloadWindows Store

Taskboards is a Trello client for Windows 8, a collaboration tool that organizes your projects into boards. It is focused on being a clean, fast and authentic way to interact with Trello.

It requires your data connection to connect to the Trello web service.

Post comments below if you have any problems, or have any recommendations for features to include. And, if you like it, post a review in the Windows Store.
